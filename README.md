# Javascript Core Concepts

This repository contains practical JavaScript examples and explanations for mastering core concepts. Each file and folder demonstrates a specific topic with code and comments.

## Folder & File Structure

- **call-apply-bind.js**: Examples of JavaScript's call, apply, and bind methods.
- **currying.js**: Demonstrates function currying using bind and closures.
- **pollyFill.js**: Shows how to create polyfills for JavaScript methods.
- **Debouncing/**: Debouncing concept with:
	- `index.js`: Debouncing logic in JavaScript
	- `text.html`: HTML to test debouncing
- **DOM/**: DOM manipulation examples
	- `dom.html`: Basic DOM operations

## How to Use

1. Clone the repository:
	 ```
	 git clone https://github.com/AkashSen03/Javascript-core-concepts.git
	 ```
2. Open files in your favorite editor (e.g., VS Code).
3. Run `.js` files with Node.js or in the browser as appropriate.
4. Open HTML files in your browser to see interactive examples.

## Topics Covered
- call, apply, bind
- Function currying
- Polyfills
- Debouncing
- DOM manipulation
